Calculating Future Value (FV)


import numpy as np

def calculate_compound_interest(principal, rate, years):
    # Standard formula for Compound Interest
    amount = principal * (1 + rate) ** years
    return amount

# Calculate for $1000 at 5% for 10 years
result = calculate_compound_interest(1000, 0.05, 10)
print(f"Future Value: ${result:.2f}")

Financial Formulas

We utilize the Net Present Value (NPV) formula:

\[ NPV = \sum_{t=1}^{T} \frac{R_t}{(1+i)^t} \]

Where:
$R_t$ = Net cash inflow-outflows during a single period $t$
$i$ = Discount rate or return that could be earned in alternative investments
$t$ = Number of timer periods
